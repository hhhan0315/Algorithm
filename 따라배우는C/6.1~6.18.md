## 6.1 while 반복 루프에서 scanf()의 반환값 사용하기
```c
#include <stdio.h>

int main() {
	int num;
	int sum = 0;
	int status;

	printf("Enter an integer (q to quit) : ");
	/*신기했다.*/
	status = scanf("%d", &num);
	// scanf의 반환값은 입력받은 변수의 개수가 몇개인지 확인 가능
	// 정수 입력 받으면 1, 문자 입력 받으면 0

	while (status == 1) {
		sum += num;

		printf("Enter next integer (q to quit) : ");
		status = scanf("%d", &num);
	}

	printf("Sum = %d\n", sum);

	return 0;
}

```
