2.9 함수 만들기
#include <stdio.h>
void say_hello(void); //prototyping, function declaration

int main(){
	say_hello();
	return 0;
}

void say_hello(void){ //function definition
	printf("hello world\n");
	return;
}
main 안에서 say_hello()가 선언만 되어있어도 사용 가능하고 컴파일이 끝나고 링커가 연결해주는 역할
